# Spotr - Workout Tracker

A modern, mobile-first workout tracking app built with React. Track your sessions, build custom programs, and share your fitness journey.

## Features

- 📱 Mobile-first design optimized for iOS
- 💪 Custom workout program builder
- 📊 Progress tracking and analytics
- 🎯 Goal setting and achievement tracking
- 🔄 Drag-and-drop program organization
- 🌙 Dark/light theme support
- 📤 Export workout data
- 🔒 Privacy-focused (no account required)

## Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Custom CSS-in-JS with design tokens
- **Storage**: Local storage with export capabilities
- **PWA**: Installable on mobile devices

## Development

```bash
npm install
npm run dev
npm run build
```

## Building for Production

```bash
npm run build
# Dist folder contains production-ready files
```

## App Store Preparation

This app is designed to be submitted to the Apple App Store via React Native or Capacitor conversion.

### Required Assets
- App icons (various sizes in `/public`)
- Screenshots (iPhone 15 Pro, various orientations)
- App Store description and keywords
- Privacy policy
- Terms of service

### Conversion Options
1. **Capacitor**: Convert to native iOS app
2. **React Native**: Rewrite for native performance
3. **Expo**: Quick deployment to App Store
